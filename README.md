# vscode-slugify

Slugify functions that used by VSCode

## Usage

```sh
npm i vscode-slugify
```

```js
import { githubSlugify } from 'vscode-slugify';

console.log(githubSlugify('Slugify functions that used by VSCode'));
// => slugify-functions-that-used-by-vscode
```
